I'm thrilled to share the launch of my latest project, the YouTube Video Transcriber. This innovative Streamlit web application is designed to transcribe YouTube videos using AssemblyAI and Google Generative AI, enabling users to dive deep into the content of videos and ask insightful questions about them.

Key Features:
Video Transcription: Automatically transcribes YouTube videos, making it easier to analyze video content.
Database Storage: Saves transcriptions in a SQLite database for future reference and analysis.
Question Answering: Utilizes Google Generative AI to provide answers to questions based on the transcribed video content.
User Interface: A sleek, user-friendly interface built with Streamlit, ensuring an intuitive experience.
Why YouTube Video Transcriber?
This application is perfect for anyone looking to analyze video content, whether for educational purposes, market research, or simply to gain insights into a topic. By transcribing videos, users can easily access the content and use it to answer questions, making the process of content analysis more efficient and accessible.

How to Get Started:
Clone the Repository: git clone https://github.com/yourusername/youtube-video-transcriber.git
Install Dependencies: pip install -r requirements.txt
Set Up Environment Variables: Create a .env file with your AssemblyAI and Google Generative AI API keys.
Run the App: streamlit run app.py
